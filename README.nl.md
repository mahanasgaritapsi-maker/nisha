# Nisha

> 🌐 Taal: Nederlands | [English](README.md) | [فارسی](README.fa.md)

Nisha is een open-source e-commerceplatform voor kleine online winkels die handmatige bankoverschrijvingen gebruiken in plaats van online betaalgateways.

## Belangrijkste functies

- Publieke winkelpagina
- Gastbestellingen en ordertracking
- Upload van betalingsbewijzen
- Dashboard voor verkopers
- Klantenportaal
- Beheerderspaneel
- RTL-interface met focus op het Perzisch

## Technologie

- Frontend: Next.js 15, TypeScript, Tailwind CSS
- Backend: FastAPI, SQLAlchemy
- Database: PostgreSQL
- Mobiele app: Flutter

## Snel starten

```bash
cp backend/.env.example backend/.env
cp frontend/.env.example frontend/.env
docker compose up --build
```
